# Getting Started

## 1. Clone the Repository

Clone the repository to your local machine using Git.

```bash
git clone https://github.com/Akira12qjw/Web-Shopee.git
```
## 2. Install Dependencies

Install the project dependencies using Yarn.

```bash
cd ShopeeClone
yarn install
```
## 3. Run 
```bash
yarn dev
```
